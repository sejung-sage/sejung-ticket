## 1. 한눈에 보기

**무엇인가요?**
세정학원 CRM의 **실제 운영 데이터**를 바탕으로, 강의실이 **시간적으로·좌석으로 얼마나 채워져 운영되는지**를 한눈에 보여주는 웹 대시보드입니다.

**왜 필요한가요?**
- 비어 있는 시간대·**저활용 강의실을 식별**해 공간 운영을 최적화
- 강의실별·기간별 **가동률/충원율**을 객관적 수치로 파악
- **미납률·출석률**까지 같은 화면에서 확인

**어떻게 접속하나요?**

| 항목 | 내용 |
|---|---|
| 주소 | **https://sejung-ticket.vercel.app** |
| 입장 코드 | 별도 전달 (내부 공유 1개 코드) |
| 사용 범위 | **대치 캠퍼스** (강의실 기록이 있는 유일한 분원) |
| 데이터 | 세정학원 CRM 운영 DB 직접 연동 · **읽기 전용** · **15분마다 자동 갱신** |

> 코드 입력 화면 뒤는 흐림 처리되어, 코드를 모르면 내용이 보이지 않습니다(내부 보안).

## 2. 화면 구성 (왼쪽 메뉴 5개)

### ① 일별 가동률 (메인)
선택한 **하루**의 강의실 현황을 표로 봅니다.
- **세로 = 강의실(29개)**, **가로 = 아침 / 오후 / 저녁** 3타임
- 각 칸에 두 가지 수치:
  - **결제** = 등록 학생수 / 정원 (예: 30/100, 30%)
  - **출석** = (등록 − 결석) / 정원 — *과거 날짜만, 미래는 빈칸*
- 색이 진할수록 충원율 높음 · 빨강은 정원 초과
- 상단 카드(세션·학생·가동률·충원율·미납률)는 **"정의" 토글**을 누르면 설명이 나옵니다
- 날짜는 달력 또는 ←/→로 이동

### ② 강의실별 가동률
선택한 **기간** 동안 강의실별 가동률을 순위로 봅니다.
- **가동률(가동일)** = 수업 있던 날만 기준 / **가동률(전체)** = 기간 전체 기준(빈 날 포함)
- "전체 가동률 낮은 순"으로 정렬하면 **평소 노는 강의실**을 바로 찾습니다
- 기간 프리셋(1·3·6개월·전체)·건물 필터 제공

### ③ 정원 관리
강의실 **물리 수용인원(정원)**을 직접 입력·수정합니다.
- 저장하면 좌석 충원율에 즉시 반영
- 현재 입시관·학종관은 정원 미입력(추후 입력 시 자동 반영)

### ④ 강좌–강의실 배정
원천 데이터에 강의실이 안 엮인 강좌를 **수동으로 강의실에 지정**합니다(보정용).

### ⑤ 시간표 업로드 ★
**매주 운영 핵심 기능.** 한글(.hwp) 시간표를 올리면 강의실 미기록 수업이 자동으로 채워집니다.
- 파일을 드래그/선택 → **업로드** 클릭 → 끝 (서버가 자동 파싱·반영)
- 하단 **달력**에 ✅(올림)/○(안 올림)으로 어느 날짜를 했는지 표시
- 자세한 운영 방법은 5장 참고

## 3. 핵심 지표 정의

| 지표 | 의미 | 계산식 |
|---|---|---|
| **세션** | 그날 열린 수업 수 | 1세션 = 강의실 × 날짜 × 강좌 |
| **학생(연인원)** | 등록 학생수 총합 | 한 학생이 N과목이면 N으로 셈(실인원과 다름) |
| **평균 가동률** | 강의실이 운영시간 중 시간적으로 얼마나 쓰였나 | Σ점유시간 ÷ Σ운영시간 |
| **좌석 가동률** | 보유 좌석-시간을 얼마나 활용했나(빈 방 포함) | Σ(학생×수업시간) ÷ (전체 정원 × 전체 운영시간) |
| **좌석 충원율** | 수업 중 좌석이 평균 얼마나 찼나 | Σ(학생×수업시간) ÷ (정원 × 수업시간) |
| **결제율(칸)** | 등록 대비 좌석 충원 | 등록 학생수 ÷ 정원 |
| **출석률(칸)** | 실제 출석 대비 좌석 충원 | (등록 − 결석) ÷ 정원 |
| **미납률** | 결제 안 한 비율 | 미납 학생수 ÷ 전체 학생수 |

> "가동률"은 **시간** 기준(얼마나 오래 쓰였나), "충원율"은 **좌석** 기준(얼마나 꽉 찼나)으로 구분합니다.

## 4. 데이터 특성 및 유의사항 (투명 공개)

신뢰도 있는 활용을 위해 데이터의 한계를 그대로 밝힙니다.

1. **강의실 미기록(29%)** — 원천 데이터의 약 29% 수업에 강의실이 비어 있습니다. **업로드한 시간표로 채워**(교사+시간 매칭) 집계에 포함합니다. 따라서 **시간표를 꾸준히 올릴수록 정확도가 올라갑니다.**
2. **출석 = 등록 − 결석** — 대치는 출석부에 **결석자만 기록**합니다. 그래서 출석 인원은 (등록 − 결석)으로 계산하며, 결석 누락 시 출석률이 다소 높게 나올 수 있습니다.
3. **미래 데이터는 희박** — 아직 진행되지 않은 날짜는 등록·출석이 적거나 비어 있고, 시간이 지나며 자동으로 채워집니다.
4. **정원 일부 미입력** — 입시관·학종관은 물리 정원 미입력 상태(좌석 지표 제외). 정원 관리 탭에서 입력 시 즉시 반영됩니다.
5. **집계 지연 15분** — 성능을 위해 집계 수치는 최대 15분 지연될 수 있습니다(개별 칸 상세는 실시간).

## 5. 주간 운영 방법 — 시간표 업로드

강의실 미기록을 채우는 **유일한 정확한 방법**이며, 담당자가 **매주 1회** 수행합니다.

1. 한 주치 시간표 한글 파일을 준비 (파일명: `요일 월_일.hwp`, 예: `월 6_8.hwp`)
2. 대시보드 → **시간표 업로드** 탭
3. 파일들(여러 개 한 번에 가능)을 선택 → **업로드** 클릭
4. 자동으로 파싱·강의실 매칭·반영 (즉시)
5. 하단 **달력에서 ✅ 확인**, 빠진 날(○)이 있으면 추가 업로드

> 같은 날짜를 다시 올리면 덮어쓰기 됩니다(안전).
